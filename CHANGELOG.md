# 更新日志

本文件记录了ArchiveBox项目的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [未发布]

### 新增
- 初始项目结构搭建
- Tauri桌面应用框架集成
- ZIP文件处理功能
- Word文档(.docx)解析和合并功能
- Excel数据导出功能
- PDF文件基础处理支持
- 现代化用户界面设计
- 跨平台构建配置(Windows/macOS/Linux)
- GitHub Actions自动构建流程
- 完整的项目文档

### 技术特性
- 基于Rust + Tauri的高性能后端
- HTML/CSS/JavaScript前端界面
- 支持拖拽式文件操作
- 实时处理进度显示
- 错误处理和用户反馈

### 依赖库
- `tauri`: 2.9.5 - 桌面应用框架
- `docx-rs`: 0.4.18 - Word文档处理
- `rust_xlsxwriter`: 0.80.0 - Excel文件生成
- `zip`: 2.4.2 - ZIP文件处理
- `lopdf`: 0.35.0 - PDF文件处理
- `serde`: 1.0 - 数据序列化
- `anyhow`: 1.0 - 错误处理

## 版本说明

### 版本号规则
- **主版本号**: 不兼容的API修改
- **次版本号**: 向下兼容的功能性新增
- **修订号**: 向下兼容的问题修正

### 变更类型
- **新增**: 新功能
- **变更**: 对现有功能的变更
- **弃用**: 不久将移除的功能
- **移除**: 已移除的功能
- **修复**: 问题修复
- **安全**: 安全相关修复
